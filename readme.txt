This is the official CoinCorner.com Magento Payment module for integration with Bitcoin

== Description ==

The CoinCorner Magento Payment module allows any site to accept payment in Bitcoin. Once connected via your API keys in your CoinCorner account, you are free to accept payments in Bitcoin and instantly be converted into GBP.

Features:

* Accept BTC payments
* Get the exact fiat amount back for your payments based on the currect price at CoinCorner.com
* Low 1% fees for accepting payments.
* Easy to use, easy to integrate, easy for your customers.

*Please note you will need an account at CoinCorner.com to use this plugin*

Once installed, your site will simply have a Pay with Bitcoin option and will be ready to go. Yay!


== Installation ==

To install, simply add this plugin to your Magento plugins and configure with your CoinCorner.com API keys.

1. Extract and upload the 'CoinCornerCheckout-Magento.zip' file to the app/code directory of your Magento installation
2. Add your API keys at Magento -> Stores -> Sales -> Payment Methods -> CoinCorner Checkout. If you do not have an account, you can sign up at CoinCorner.com
3. Activate the plugin via Magento -> Stores -> Sales -> Payment Methods -> CoinCorner Checkout
